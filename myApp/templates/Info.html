Método del punto fijo 😬

¿En qué consiste el método de punto Fijo?

Es el método que se encarga de buscar la raíz de una función a partir de un valor inicial, una margen de error y un numero "n" de iteraciones. Para tener en cuenta, en este método no es necesario el uso de intervalos.

Requerimientos empleados en este programa:
  • Función F(x). Ecuación a despejar y para obtener gráfico.
  • Función G(x). Ecuación utilizable para determinar si es una función catalogada como G(x). Los requisitos son que, al derivarse y evaluarse la derivada de la misma, debe de estar entre 0 y 1 para que sea una buena G(x).
  • Valor Inicial: Es el que se otorga para poder evaluar e iniciar las iteraciones.
  • Margen de error: Será el porcentaje que determine que tan cercana esta nuestra raíz a la real.

Python 🐍

¿Qué es Python?

Python es un lenguaje de programación creado por Guido van Rossum a principios de los años 90 cuyo nombre está inspirado en el grupo de cómicos ingleses “Monty Python”.Este se trata de un lenguaje interpretado o de script, con tipado dinámico, fuertemente tipado, multiplataforma y orientado a objetos y el cual ha sido empleado para la realización de este proyecto.

¿Por qué usar Python y no otro lenguaje?

Para el proyecto se ha hecho uso de Python debido a su sintaxis simple, clara y sencilla tomado en cuenta que este posee de una gran cantidad de librerías disponibles que se adaptan muy bien a las necesidades que se tienen a la hora de programar el método antes mencionado.

Django 😎

¿Qué es django?

Django es un framework web de alto nivel de Python que fomenta el desarrollo rápido y el diseño limpio y pragmático. Construido por desarrolladores experimentados, se encarga de gran parte de las complicaciones del desarrollo web, para que quien lo usa pueda centrarse en escribir su aplicación sin necesidad de reinventar la rueda.
